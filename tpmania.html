TpMania

SYSTEM ID: GTR-HERO-STM32
PROJECT BRIEF

A rhythm-based embedded game system inspired by Guitar Hero. The system processes user inputs in real-time to match scrolling notes displayed on an LCD, requiring precise timing and interrupt handling. Handles audio through the use of audio DAC and a stereo amplifier. This entire project runs of a libo battery with built in charging when plugged into a power source.

TECHNICAL SPECIFICATIONS
MCU: STM32 (ARM Cortex-M) LANGUAGE: C / Python DISPLAY: SPI OLED Display INPUT: 4x Tactile Buttons (GPIO IRQ) PROTOCOL: SPI, I2C, UART
ENGINEERING CHALLENGES

The main challenge was learning new pcb manufacture techniques to ensure no noise or arrors when transmitting high frequency signalys.
