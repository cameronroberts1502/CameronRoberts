CAM_02: PCB_TOP_LAYER

Loggy

SYSTEM ID: DATA-LOGGER-AVR
PROJECT BRIEF

A multi-sensor data acquisition unit designed for lab environments. It simultaneously samples voltage, acceleration (IMU), and temperature, logging the data to an SD card while displaying real-time metrics on an LCD.

TECHNICAL SPECIFICATIONS
MCU: ATmega328P (AVR) SENSORS: MPU6050 (Accel), ADC (Volt) STORAGE: SD Card (SPI Mode) POWER: Linear Regulated 5V/3.3V DISPLAY: 16x2 Character LCD
ENGINEERING CHALLENGES

Implemented a custom file system buffer to minimize SD card write latency. Designed a custom PCB with separated analog/digital ground planes to reduce noise on the voltage ADC readings.
